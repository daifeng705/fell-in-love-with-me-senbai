# 表面高冷实际傲娇的艺联领队想和我谈恋爱

校园二次元文字互动选择类游戏 — 以高校艺术联队辩论队为背景，双数值养成、多分支剧情、多结局解锁的轻量化互动网页游戏。

## 游戏简介

玩家以大学新生视角进入剧情，通过不同对话选择改变核心数值（好感度、了解度），解锁专属剧情片段，最终达成差异化结局。

**核心人设：** 男主「椰子饭」— 打破高冷学长刻板印象，外在严谨犀利、严苛毒舌的辩论队领队，内在是腼腆纯情、痴迷二次元的傲娇少年。副线男主「猫猫坟」主打温柔治愈。

## 剧情阶段

六大剧情阶段层层递进，每个阶段设置多分支选择：

| 阶段 | 说明 |
|------|------|
| 初识 | 与辩论队的初次相遇 |
| 靠近 | 逐渐熟悉彼此 |
| 深入 | 关系进一步深化 |
| 裂痕 | 误会与冲突 |
| 转折 | 关键抉择时刻 |
| 终局 | 结局解锁 |

## 结局一览

根据数值累计、互动次数、终局选择，解锁四种独立结局：

- **甜蜜恋爱** — 好感度与了解度双高，达成完美告白
- **挚友同好** — 了解度高但好感度不足，成为知心好友
- **温柔移情** — 副线互动充分，走向猫猫坟线
- **遗憾陌路** — 数值不足，擦肩而过

## 玩法系统

- **双数值驱动**：好感度（心动值）+ 了解度（认知值），数值配比决定最终结局
- **分支递进**：每个选项包含正向加成、踩雷扣分、隐藏解锁、剧情逆转
- **细节彩蛋**：矿泉水、圣斗士、假面骑士、手办等专属暗号与二次元同好彩蛋
- **保存进度**：支持手动存档/读档与自动存档，随时继续游戏

## 技术架构

```
AI 素材生成  →  AI 文案创作  →  OpenCode 网页开发
```

| 模块 | 技术实现 |
|------|----------|
| 前端框架 | 原生 HTML + CSS + JavaScript |
| 开发工具 | OpenCode 在线代码编辑器 |
| 视觉素材 | AI 图像生成（角色立绘、场景背景、结局封面） |
| 文案创作 | 豆包 AI 辅助完成剧情框架、分支逻辑、数值体系 |
| 音频系统 | 标题画面 BGM + 游戏内 BGM + 点击音效 |
| 存储系统 | localStorage 存档/读档、设置持久化 |

## 特性

- 粉色调日系视觉风格，樱花飘落 + 星光粒子动态效果
- 圆形菜单面板设计，全响应式布局（桌面/平板/手机）
- 实时设置（文字速度、BGM 音量、SFX 音量、亮度调节）
- CG 回想画廊，解锁后可随时回顾
- 浏览器直接游玩，无需下载安装

## 开始游玩

直接打开 `index.html` 即可开始游戏，或部署到任意静态托管服务。

## 开发

项目包含三个核心文件：

- `index.html` — HTML 结构与布局
- `style.css` — 全部样式与动画
- `script.js` — 游戏逻辑与数据
